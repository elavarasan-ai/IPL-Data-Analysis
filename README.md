# 🏏 IPL Data Analysis Using Python

A complete end-to-end Exploratory Data Analysis (EDA) project on 12+ seasons of Indian Premier
League (IPL) cricket data — from raw match/ball-by-ball CSVs to cleaned data, 20+ analyses,
visualizations, and business insights.

---

## 📌 Description

This project analyzes IPL `matches.csv` and `deliveries.csv` datasets to answer real
cricketing and business questions: which teams and players dominate, whether winning the toss
matters, which venues favor chasing, and how scoring trends have shifted across seasons. The
project follows a professional data analytics workflow: **load → clean → engineer features →
explore → visualize → conclude.**

---

## ✨ Features

- Clean, documented Jupyter Notebook with 13+ structured sections
- Full data cleaning pipeline (missing values, duplicates, team-name standardization, date
  conversion)
- 20+ exploratory analyses (team performance, player stats, toss impact, venue trends, Orange
  Cap / Purple Cap analysis, and more)
- 5 chart types (bar, line, pie, histogram, scatter) each with a "why / what / how to read"
  explanation
- Business-question-driven insights, not generic observations
- Beginner-friendly, PEP 8-compliant code with inline comments

## 📊 Dataset

- **Source:** [IPL Complete Dataset (2008–2020) — Kaggle](https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020)
- **Files used:** `matches.csv` (match-level data), `deliveries.csv` (ball-by-ball data)
- Download the CSVs and place them inside `data/raw/` before running the notebook.

> ⚠️ Column names can vary slightly between dataset versions (e.g. `batsman` vs. `batter`).
> The notebook checks `.columns` early on — update any column name references if your file
> differs.

## ▶️ Usage

1. Download `matches.csv` and `deliveries.csv` from the [Kaggle dataset link](https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020) and place them in `data/raw/`.
2. Open the project in VS Code: `code .`
3. Open `notebooks/IPL_Data_Analysis.ipynb` and select the `venv` kernel.
4. Run all cells from top to bottom (`Run All`).

## 📈 Results

- Identified the most successful IPL teams by total wins **and** win percentage.
- Quantified the real (often overstated) impact of winning the toss on match outcome.
- Ranked top batsmen and bowlers by runs, wickets, strike rate, and economy rate.
- Surfaced venue-specific batting-first vs. chasing trends.
- Tracked season-over-season scoring trends across 12+ IPL seasons.
